Sigma Rules

Sigma is an open standard for writing detection rules for SIEMs.

Example Rule: Suspicious Base64 Execution

title: Suspicious Base64 in Command Line
logsource:
  category: process_creation
detection:
  selection:
    CommandLine|contains: "base64"
  condition: selection
level: medium

This detects base64 usage in command lines, which is common in obfuscated attacks.
